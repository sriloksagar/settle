# Splitwise-like App Prototype

This project is a prototype of a Splitwise-like app built with React and Tailwind CSS.

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm start` to start the development server
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Deploying to GitHub Pages

1. In `package.json`, add a `"homepage"` field with your GitHub Pages URL:
   ```json
   "homepage": "https://<username>.github.io/<repository-name>"
   ```
2. Run `npm run deploy` to deploy the app to GitHub Pages
